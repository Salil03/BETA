## Installation Instructions


Create input files in the format `1.in, 2.in,3.in ....` and expected output files as `1.out, 2.out, 3.out...`

Install easygui and prettytable using `pip install easygui` and `pip install prettytable`. Run `test.py` and select source code, input files, output files and see the verdict. 

AC = Accepted
WA = Wrong Answer
